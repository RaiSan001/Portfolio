# 🛡️ Cyber Portfolio — Sanskar Bhattarai

A personal cybersecurity portfolio and research hub, showcasing CTF writeups, vulnerability analyses, study notes, and policy papers. Built as a static site with **Next.js (App Router)**, **MDX**, **Tailwind CSS**, and deployed on **Cloudflare Pages**.

🔗 **Live site:** https://sanskarbhattarai.com.np

---

## ✨ Features

- **MDX papers** – Write rich, technical content with markdown + React components.
- **PDF downloads** – Attach supplementary PDFs to any paper.
- **Fully responsive** – Works on desktop, tablet, and mobile.
- **Smooth scrolling** – Anchor links for sections (About, Skills, Projects, Contact).
- **Dark theme** – Optimised for readability.
- **Static export** – Lightning‑fast, secure, and cheap to host.

---

## 🧰 Tech Stack

| Category       | Tools |
|----------------|-------|
| Framework      | Next.js (App Router, static export) |
| Styling        | Tailwind CSS + Tailwind Typography |
| Content        | MDX, gray-matter, next-mdx-remote |
| Icons          | lucide-react |
| Deployment     | Cloudflare Pages (`_headers`, `_redirects`) |
| Code quality   | TypeScript, ESLint |
